● This is an expense tracker web application called "Spendly" - a student
  project built with Flask. Here's what it does:

  Current State

  The project is a work-in-progress Flask web app with:

  Implemented Features:

  - Landing page - Homepage for the app
  - User registration - /register route with template
  - User login - /login route with template
  - SQLite database setup (in database/db.py) - ready for student implementation

  Planned Features (not yet implemented):

  - User logout
  - User profile page
  - Add expenses
  - Edit expenses
  - Delete expenses

  Tech Stack:

  - Backend: Flask 3.1.3
  - Database: SQLite (via database/db.py)
  - Frontend: HTML templates with CSS/JS
  - Testing: pytest + pytest-flask

  This appears to be a campus/educational project (based on the repo name
  spendly_by_campusX) where students learn to build a full-stack expense
  tracking application by implementing features step-by-step.
